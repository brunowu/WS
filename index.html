What is SMG2S ?

Motivation

Nowadays, IT areas like machine learning and artificial intelligence use iterative linear algebra methods. Algorithms are increasingly using matrices which had never been than large. In order to decrease computing time, we have to find non-Hermitian solvers. They will allow to find a answer to eigenvalues and linear problems on parallel and/or distributed machines.

Method

The inputs of this program are Matrix Market Vector Format files. Starting from given spectra, SMG2S uses eigenvalue to generate large-scale non-Hermitian matrices. It’s a matrices generation method by selecting many parameters that gives one with a high level of accuracy.

Parallel calculations

SMG2S can be implemented both on clusters of CPUs and GPUs. It provides several methods to verify the generated matrices. Based on an open source parallel software, the algorithm optimazes basic operation to decrease computing time.

UI

You can use the graphical user interface to check the algorithm’s ability to keep the given spectra.

How does the UI work ?

Select your files

After generating a matrix, you can select the original and the final files to start the verification.

Display your results

You can examine your graphic directly on the application. Moreover, it is possible to open as many graph as you want to compare them.

Examine each result

With your UI, you can move or zoom your graphic to avoid missing any information.

Save your own graphic

You can create the perfect graphic with your custom settings.
